## 使用指南

### 依赖检查

- [Ren'Py Launcher](https://www.renpy.org/latest.html)（建议使用 Ren'Py 8.x）
- 若缺少其他运行依赖库，请根据日志信息或 Ren'Py Launcher 的提示补充下载。

### 构建

本项目通常不需要“构建”步骤。将本项目文件夹存放在 Ren'Py Launcher 的项目目录下即可。

### 运行

在 Ren'Py Launcher 中选择本项目并点击“启动项目”。

## 如何贡献

欢迎任何形式的贡献：剧情提案、BUG 修复、立绘与音乐素材、脚本与翻译。

贡献流程建议:

1. Fork 仓库并创建相应分支。
2. 使用 Issues 报告 BUG 与功能建议，提交清晰的 commit 信息并在 PR 描述中说明变更点。
3. 若添加第三方素材，请在 PR 中附上授权说明（来源与许可），不得提交有版权争议的商业素材。
